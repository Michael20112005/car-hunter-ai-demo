# Car Hunter AI — public demo

Це статична публічна демо-версія для GitHub Pages.

## Публікація через GitHub Pages

1. Створи публічний репозиторій, наприклад `car-hunter-ai-demo`.
2. Завантаж у корінь репозиторію:
   - `index.html`
   - `.nojekyll`
   - `robots.txt`
3. Відкрий `Settings` → `Pages`.
4. У `Build and deployment` вибери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Збережи налаштування.
6. Отримаєш адресу на кшталт:
   `https://YOUR_GITHUB_USERNAME.github.io/car-hunter-ai-demo/`

## Безпека

Не додавай у публічний репозиторій `.env`, API-ключі, Telegram token/chat ID або SQLite-базу.
