Car Hunter AI

Car Hunter AI — персональний сервіс для автоматизованого моніторингу оголошень про продаж вживаних автомобілів в Україні.

Система створюється для збору оголошень, базової фільтрації, дедуплікації, оцінювання ризиків та надсилання найцікавіших варіантів мені.

Статус проєкту: публічна демонстраційна версія.
Дані автомобілів у цьому репозиторії є тестовими.
